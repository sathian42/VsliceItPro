📽️ Welcome to Vsliceit
Vsliceit is a high-speed, privacy-focused video automation tool designed to help you split long movies into perfect social-media-ready clips. Built with the power of FFmpeg.WASM, it handles all the heavy lifting directly on your computer—no uploads, no data costs, and total privacy.

🌟 Key Features
Smart Omission: Automatically skip intros, credits, or unwanted scenes by simply entering the time ranges.

Browser-Side Processing: Your videos never leave your device. We use your hardware to encode, ensuring 100% data security.

Auto-Download Engine: No more fighting with folder permissions. Clips are generated and saved to your default Downloads folder instantly.

Multi-Format Support: Choose between 9:16 Vertical (perfect for Reels/TikTok) or 16:9 Wide (YouTube standard).

Dynamic Overlays: Automatically adds "Part #" and Movie Titles to your clips so they are ready for posting immediately.

🛠️ How to Use Vsliceit
Enter Movie Details: Type the name of the film for the bottom overlay.

Set Your Splits: Decide how many seconds each part should be (default is 60s).

Define Omit Zones: Paste the timestamps you want to skip (e.g., 00:00:00 - 00:00:20).

Upload & Go: Select your video file and click Start.

Check Downloads: As each part finishes, it will appear in your browser's download bar.

🔒 Security & Privacy
Because Vsliceit runs using WebAssembly (WASM), it functions like a desktop app inside your browser.

No Server Storage: We don't have a server to store your videos.

Zero Account Required: No sign-ups, no tracking.

Open Source Logic: Built by a developer for developers and content creators.
